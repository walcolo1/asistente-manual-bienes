# Base Rules

## Cuándo usarla
Siempre que se trabaje en la arquitectura base del proyecto, configuración de servidores o inicialización de frameworks.

## Contexto del proyecto
El proyecto consta de un backend en Node.js (Express) y un frontend en React (Vite).

## Instrucciones obligatorias
1. Mantener la separación de responsabilidades entre frontend y backend.
2. Utilizar el puerto 3000 para el backend y 5173 para Vite por defecto, a menos que se indique lo contrario.
3. No modificar los archivos marcados como protegidos en la directiva principal (AGENTS.md, README.md, .gitignore, etc.).

## Restricciones
- No instalar dependencias innecesarias o herramientas no autorizadas en este stack.
